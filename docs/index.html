这个项目是做什么的？

第一，反“AI 引战型低质内容”。 B 站、知乎、小红书上很多打“资本与劳动对竟”、“生育下跌”、“互联网院薪低”的内容， 很多是抽了部分实数据上口头，并未给出原始来源与口径。 本页面仅采纳《国家统计年鉴》、《BLS 年报》、《IMF WEO》《ILO WESO》等可直接验证的原始公开文件，并在每张图表下方标注。

第二，跨国别、跨岗位、跨时间看中、美、世界的不平衡。 居民可支配收入、人均 GDP、CPI、房价、失业率、基尼系数、劳动分配率以及“农民 / 都市白领 / 计算机行业 / AI 研究员”这几个岗位同时可观。其中“都市白领”不是国家统计口径，本页面以《城镇非私营单位平均工资》为上限估算（含企事业单位、院校、金融、IT 等都市岗位），以《城镇私营单位平均工资》为下限估算（含小微企业、服务业、个体商户为主），两者中间才是“白领”实际所在。

第三，数据本身可被公开审计。 每个指标带 source method confidence 三个标签。 official = 可以在原始部门网站验证； academic = 依赖学术口径估算； industry_sample = 行业平台报酬样本。

你可以点击每张图表下方的「原始来源」直接跳转到发布页。 如果发现与官方公开数据不一致，请提 PR；最危险的是“为了引战而改动定中中间资料”。
